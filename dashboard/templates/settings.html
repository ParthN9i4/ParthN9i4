Setup Guides

Notion Setup
  1. Go to notion.so/my-integrations
  2. Create a new integration and copy the API key
  3. Create two databases in Notion (Events and Todos)
  4. Share each database with your integration
  5. Copy each database ID from the URL (the 32-character string after the workspace name)
  6. Paste the values above and save
Discord Webhook Setup
  1. Open Discord and go to your server settings
  2. Go to Integrations > Webhooks > New Webhook
  3. Name it "PPML Dashboard" and choose a channel
  4. Copy the webhook URL and paste it above
Gmail SMTP Setup
  1. Enable 2-Step Verification on your Google Account
  2. Go to Google Account > Security > App Passwords
  3. Generate an app password for "Mail"
  4. Use smtp.gmail.com, port 587, your Gmail as username, and the app password
Obsidian Setup
  1. Find your Obsidian vault directory (the folder containing your .obsidian folder)
  2. Enter the absolute path above
  3. Click "Export to Obsidian" to create markdown notes
  4. A "PPML-FHE-Dashboard" folder will be created inside your vault
